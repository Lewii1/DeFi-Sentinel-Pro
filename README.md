# DeFi Sentinel Pro (Apex Predator Edition)

DeFi Sentinel Pro is an Institutional-Grade Cyber Terminal built for deep-liquidity yield farming analysis. Designed for professional operators, it aggregates, filters, and formats real-time on-chain data directly from the DefiLlama API. 

The "Apex Predator Edition" strips away unnecessary interface elements to maximize data density (V21), highlighting maximum ROI environments across multiple blockchains simultaneously.

## Core Features

- **Deep Liquidity Matrix**: Strictly filters and monitors protocols with a Total Value Locked (TVL) > $10,000,000 to filter out micro-cap noise.
- **Apex Hunter Mode**: A background recursive scanning algorithm. If a monitored asset bridges >1,000% APY, the Terminal isolates it with a pulsing neon perimeter, triggers a local radar audio ping, and fires a native Desktop Notification globally.
- **Whale UI (Golden ROI Luminous)**: Input an active capital allocation. If the algorithmic projection calculates your 12-month Annual Return physically eclipsing your initial allocation sum, the interface mutates the value projection into Neon Gold.
- **Dynamic Candlestick Charts**: Processes 7-day trajectory volatility by replacing traditional tracking lines with live green/red SVG block candlesticks.
- **Triple-Layer Node CDN**: Complete visual resilience. Parses the LlamaFi network icons, failing back autonomously to TrustWallet GitHub repos and CryptoIcons APIs, executing a native CSS textual icon if all global networks drop.
- **Asymmetric Auto-Sync**: The terminal limits excessive API flooding by pacing autonomous background data dumps strictly to 300-second (5-minute) intervals, traced by a hyper-narrow top-level loading element.
- **Gas Tracker**: Live tracking footer for Ethereum, Solana, Arbitrum, Base, BSC, and Polygon baseline transaction cost averages.

## Installation

You must have [Node.js](https://nodejs.org/) installed on your operating system.

1. Clone or download this repository.
2. Open your terminal in the root directory of the project.
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Boot Methods

### Windows
Simply double-click the `start_windows.bat` file located in the root directory.

### Linux / Mac
1. Open terminal in the directory.
2. Run the Node server natively:
   ```bash
   node server.js
   ```

The terminal will automatically bind to your local port. Open your browser and navigate to:
`http://localhost:3000`

## Export Architecture
All internal API routing variables (`/api/yields`) are built fully relatively. The frontend architecture scales to 98% Ultra-Wide viewing dimensions safely and can be containerized or port-forwarded without breaking external API CORS mapping. You can dump the active local states asynchronously by clicking the `EXPORT CSV` ghost button on the bottom right footer.
